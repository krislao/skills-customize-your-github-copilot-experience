# 📘 Assignment: Testing with pytest

## 🎯 Objective

Learn how to write automated tests in Python using `pytest` and verify small functions with assertions.

## 📝 Tasks

### 🛠️	Write tests for basic functions

#### Description
Implement tests for existing utility functions and confirm they return the expected results.

#### Requirements
Completed program should:

- Include a `tests/test_math_utils.py` file with tests for arithmetic helper functions.
- Use `assert` statements to verify correct behavior.
- Test edge cases such as zero, negative numbers, and invalid input where appropriate.

### 🛠️	Run pytest and interpret results

#### Description
Run the test suite and inspect pytest output to understand passing and failing tests.

#### Requirements
Completed program should:

- Execute `pytest` successfully against the `tests/` directory.
- Display a summary of passed tests.
- Fix any failing tests so that all tests pass.
